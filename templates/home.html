ML-Powered CRISPR Analysis

The year 2001 saw the first rendition of the Human Genome Project with the first human genome sequence being fully sequenced in 2003. The project induced a great sense of optimism for many researchers, as genomic medicine was considered a pivotal cornerstone for medical diagnosis and treatment. The need for a cost-efficient means of analysing the large amount of genomic data in a meaningful way has been requested for a long time. Genomic medicine and ML have been the core of many publications by Nvidia and IBM. With these tools this WebApp has been created to aid researchers in  predicting cut site sizes from a given sequence.


Insertion/Deletion Prediction for Sequence Data

This webapp will predict insertion/deletion spread of single strand DNA sequences using data provided by the Wellcome Sanger Institute. Currently limited to a 52 base sequence containing the CRISPR from bases 29 to 48 and the PAM from bases 49 to 51. Simply type or paste your sequence below and click Predict. You can test the server using this sample sequence.
